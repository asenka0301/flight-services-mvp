## [Flight Services MVP Live Demo](https://flight-services-mvp.vercel.app/)

Небольшой экран для MVP сервиса дополнительных услуг для авиапассажиров.

## Что реализовано

- Список услуг
- Для каждой услуги: название, цена, возможносить добавить в список выбранных
- Блок **“Итого”**:
  - список выбранных услуг
  - общая сумма
  - кнопки “Оформить заказ” (заглушка) и “Очистить”

## Стек

- React 18
- TypeScript
- Vite
- CSS Modules

## Запуск проекта

```sh
$ git clone https://github.com/asenka0301/flight-services-mvp.git
$ cd flight-services-mvp
$ npm i
$ npm run build
$ npm run start
```
